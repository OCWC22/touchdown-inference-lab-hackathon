# Blog mirror

`qwen36-openclaw-mlx-macbook-kv-cache.html` is a local mirror of the Touchdown Labs post
[Set Up a Local OpenClaw or Hermes KV-Cache Learning Loop with Mac MLX](https://touchdown-labs.com/blog/qwen36-openclaw-mlx-macbook-kv-cache.html).

It is included here as the take-home extension of the lab: after the three-minute lesson, run the same
KV-cache/token discipline on your own Mac. See [`docs/token-optimization.md`](../docs/token-optimization.md)
for how it connects to the frozen task, the agent-optimizer skill, and the receipt schema.

The canonical copy lives on touchdown-labs.com; image assets are mirrored alongside and root-absolute
links were rewritten so the page works when served from this repo (`npm run dev`, then `/blog/...`).
