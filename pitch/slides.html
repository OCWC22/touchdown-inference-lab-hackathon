Slide 02 / Product Overview

Run one coding task. See every token the agent remembered, recomputed, and wasted — then fix it.

The pain

Coding agents hide the bill inside the loop. One small retry-bug fix loaded 40,176 characters of harness context — 17,420 of them repeated (43%) across 14 tool calls. Students see a patch; nobody teaches them what the system paid, or how to make it cheaper without breaking the tests.

The built solution

A browser lab on a frozen, reproducible coding task. A three-minute, nine-stage lesson: classify context lifetime, apply one optimization policy at a time, watch the engine decision flip from local MLX-LM to vLLM on Nebius at 100 users, follow text → tokens → prefill → KV cache → HBM, and export a signed optimization receipt. EverOS memory makes the next lesson adapt; Butterbase persists the evidence.
