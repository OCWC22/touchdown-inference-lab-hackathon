# Touchdown Inference Lab demo videos

## touchdown-inference-lab-lifecycle-demo.mp4 (primary)

A 1:48, browser-only walkthrough of the complete lifecycle across all three surfaces:

1. `index.html` — the principle, the loop bar, the full-stack strip, then all nine live lesson
   stages with every interaction exercised: context-lifetime controls toggled, the engine
   decision flipped to one hundred users, all three sponsor operations run, recalled memory
   applied, the result verified, and the receipt persisted through the offline queue.
2. The evidence sections — challenge ladder, résumé-vs-receipt comparison, and the
   incentive-alignment thesis.
3. `lab.html` — the thirteen-phase continuous lab, pausing on the GPU/HBM package schematic
   and the facility (power / thermal / cost) lanes.
4. `journey.html` — the learner-to-contributor loop, story steps, L1–L9 ladder, and the
   rendered portfolio receipt, closing on the final principle.

Media details: 1280 × 720 · H.264 · 30 frames per second · no audio · 1:48 runtime
(under the 2:00 cap). Contact sheet: `lifecycle-contact-sheet.jpg`.

Recorded with a scripted Playwright walkthrough against the local build; every interaction
shown is the real application code running on the deterministic fixture.
