# Touchdown Inference Lab demo

`touchdown-inference-lab-browser-demo.mp4` is a 43-second, browser-only walkthrough of the hackathon demo.

The recording shows the pinned coding task, context optimization, local-to-production engine selection, sponsor architecture, the self-hosted inference path, EverOS adaptive memory, verification, and the contribution path.

Media details:

- 1280 x 720
- H.264
- 30 frames per second
- No audio

Evidence boundary: the walkthrough uses the demo's explicitly labeled deterministic fixture states. It does not present fixture data as a live production run.
